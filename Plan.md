## Functionality
1. **Get**
   * Scrape from IMDb using OMDb API and Beautiful Soup
     - Overall user ratings
     - My user ratings
     - Box office movies from favorite theater
2. **Evaluate**
    * Feed data into training model
3. **Return**
    * Send notification when interesting movie found
